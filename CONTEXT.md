ПРОЕКТ: Malware Confinement — Музыкальный плеер на PyQt6

Что это: Десктопный музыкальный плеер с красивым UI, 15 темами, онлайн-стримингом и GIF-панелью. Собран в .exe через PyInstaller. Репозиторий: https://github.com/gdaf5/Player

Технологии: Python 3.13, PyQt6, yt-dlp, mutagen, pycaw, PyInstaller

Структура файлов:
- main.py — весь код (~2200 строк, один файл)
- icon.ico — иконка приложения
- MalwareConfinement.spec — конфиг для PyInstaller
- .gitignore — исключения
- gifs/ — папка с GIF-анимациями для правой панели
- downloads/ — папка для скачанных треков (создаётся автоматически)

Реализованные функции:
1. Локальная музыка — динамическое сканирование выбранной папки (рекурсивно), чтение ID3 тегов (MP3, M4A, FLAC, OGG, WAV), извлечение обложек из тегов и файлов папки
2. Онлайн-поиск — YouTube и SoundCloud через yt-dlp, режим URL (вставляешь ссылку — играет)
3. Скачивание — треков из поиска в папку downloads/
4. 15 тем — Midnight, Ocean, Sunset, Forest, Light, Cyberpunk, Rose Gold, Nord, Cherry Blossom, Deep Purple, Emerald, Amber, Crimson, Aqua, Spider
5. GIF-панель — справа, рандомная гифка при каждом треке
6. Плейлисты — создание, удаление, добавление треков (ПКМ)
7. Управление — play/pause, prev/next, shuffle, repeat (off/all/one), громкость + mute
8. Выбор аудио-устройства — QComboBox с системными выходами, переключение без остановки
9. Эквалайзер — UI диалог с 10 полосами (32Hz-16kHz), пресеты (Rock, Pop, Jazz и т.д.), сохранение состояния. Кнопка в панели управления
10. Настройки звука — кнопка открывает mmsys.cpl
11. Бегущая строка — для длинных названий треков
12. Поиск — фильтрация локальных треков
13. Автосохранение — config.json в %USERPROFILE%\.MalwareConfinement\
14. Синхронизация системной громкости — через pycaw, изменение громкости в плеере меняет системную громкость Windows

Как запустить:
  pip install PyQt6 mutagen yt-dlp pycaw
  python main.py

Как собрать .exe:
  pip install pyinstaller
  pyinstaller MalwareConfinement.spec

Конфиг хранится в: %USERPROFILE%\.MalwareConfinement\config.json
Пути: SCRIPT_DIR определяет папку приложения (работает и из .exe, и из .py), DATA_DIR = ~/.MalwareConfinement

Известные моменты:
- Эквалайзер — полная интеграция с Equalizer APO. Автоопределение установки, генерация конфига, реальное влияние на звук
- Обложки: извлекаются из ID3 тегов (APIC для MP3, covr для M4A, pictures для FLAC) + поиск folder.jpg/cover.jpg в папке
- Ни один из 366 тестовых MP3 файлов не имел встроенных обложек — это нормально, зависит от файлов пользователя
- Переключение аудио-устройств: плавное переключение с сохранением позиции воспроизведения
- Синхронизация громкости: изменение громкости в плеере автоматически меняет системную громкость Windows через pycaw (работает для дефолтного устройства Windows)
